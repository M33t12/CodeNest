Module : User module
I want role based access : Which user have which permissions ? 
Two roles :

User : Normal User
Admin : Specialized user :

What user can do ? 
-- user can upload resources , view resources upload by other users:
-- Resources have type : [pdf, link , image, text-if user want to add , blogs]

What Admin can do ?
For Resources that are added by any users , first goes to the admin to verify the resource, if resource permits then the resource will be visible in website
Admin dashboard have all users data like [no of users, open the profile of each user , view each user's profile---> contains how many resources are uploaded by users, and also view particular resource]
-Admin can add other admins
-admin have the rights to block other users also...
-admin have all the information of all users of website

How admin verify the resources ::
1. user upload the resource.
2. Resource is sent to the admin for verification.
3. Admin can verify the resources by Ai-Model, Ai-model give verdicts based on the authenticity of resource,subject-area,is it useful or not
4. Based on the Ai-model verdicts Admin can approve the resources/reject the resources.
5. after approval it will be displayed to all the users in resources section of website.
